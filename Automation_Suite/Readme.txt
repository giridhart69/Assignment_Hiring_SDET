Configs: Stores configuration settings like browser_configs, driver_configs in formats like ini, yaml, json.
Utils: Contains reusable helper functions and classes.
Pages: Implements the POM(page object Model) with each file representing a page and containing element locators and specific actions related to the page.
tests: Holds the actual test scripts, with files containing test classes and individual test methods with assertions.
main.py: This is the entry point to run tests.
